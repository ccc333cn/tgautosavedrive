## v0.0.1 — 2026-04-29

首个完整版本，Go 重构全部功能实现。

### 核心功能

- Telegram 频道搜索：HTTP 抓取 + MTProto 双模式，多频道并行搜索，全量/增量扫描，本地缓存。
- 115 网盘集成：扫码登录、文件浏览、资源转存、云下载。
- 自动追更：Cron 定时任务，按剧名/季自动整理目录结构，去重转存，超期自动完结。
- 智能解析：支持 Case A/B/C/D 四种频道格式，自动提取元数据并按画质保留最优资源。

### 联动推送

- SmartStrm：转存后触发 Webhook 生成 STRM 文件，按类型去重。
- Emby：SmartStrm 通知后延时触发媒体库扫描。
- 通知：PushPlus 微信 + Telegram Bot 双通道，30s 防抖合并推送。
- 任务完结推送：过期自动完结时推送通知。

### 前端与系统

- Vue 3 单文件 SPA，Obsidian Dark 主题，Lucide Icons SVG 图标系统。
- 转存概览、资源搜索、网盘浏览器、频道管理、豆瓣榜单、日志、记录、系统设置等页面。
- 数据备份恢复、日志自动清理、缓存自动刷新、Bearer Token 认证、PWA 支持。
