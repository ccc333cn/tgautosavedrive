  • 修复 123 追更季文件夹重复创建报错的问题（改用 FindOrCreateFolder）。
  • 新增 123 追更转存前网盘文件名去重，避免重复转存相同文件。

  • 云下载标题添加"仅 115 网盘"标签提示。

v0.0.1 2026-04-29 首个版本

首个完整版本，Go 重构全部功能实现。

核心功能
  • Telegram 频道搜索 — HTTP 抓取 + MTProto 双模式，多频道并行搜索
  • 115 网盘集成 — TLS 指纹绕过 WAF，扫码登录，文件浏览/转存/云下载
  • 自动追更 — Cron 定时任务，按剧名/季自动整理目录，三重去重
  • 智能解析 — Case A/B/C/D 四种频道格式，自动提取元数据
联动推送
  • SmartStrm — 转存后触发 Webhook 生成 STRM 文件
  • Emby — SmartStrm 通知后延时触发媒体库扫描
  • 通知 — PushPlus 微信 + Telegram Bot 双通道，30s 防抖合并推送
前端
  • Vue 3 单文件 SPA，Obsidian Dark 主题，9 个导航页
  • Lucide Icons 全量 SVG 图标系统
  • 仪表盘、资源搜索、网盘浏览器、频道管理、豆瓣榜单等
系统
  • 数据备份恢复、日志自动清理、缓存自动刷新
  • Bearer Token 认证、PWA 支持
技术栈
  • Go 1.25 · Gin · gocron/v2 · GORM + SQLite（modernc，无 CGO）
  • Vue 3 CDN · Lucide Icons · embed.FS
  • Docker Alpine 单容器 · 端口 39977
